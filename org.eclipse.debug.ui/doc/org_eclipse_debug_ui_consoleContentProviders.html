Console Content Providers

Identifier: org.eclipse.debug.ui.consoleContentProviders

Description: This extension point provides a mechanism for contributing a content provider for a process. The content provider will be used to produce and render output in the console.

Configuration Markup: 

   <!ELEMENT consoleContentProvider>
   <!ATTLIST consoleContentProvider 

      id            CDATA #REQUIRED

      class         CDATA #REQUIRED
  • id specifies a unique identifier for this console content provider.
  • class specifies a fully qualified name of a Java class that implements IConsoleContentProvdider.

Examples:

The following is an example of a console content provider extension point: 

<extension point="org.eclipse.debug.ui.consoleContentProviders">
  <consoleContentProvider 
     id="com.example.ExampleConsoleContentProvider"
     class="com.example.ExampleConsoleContentProvider">
  </consoleContentProvider>
</extension>

In the above example, the contributed content provider will be use for processes that refer to this content provider's id. A process refers to a console content provider by specifying a console content provider's id in the process attribute table, keyed by IConsoleContentProvider.ATTR_CONSOLE_CONTENT_PROVIDER.

API Information: Value of the attribute class must be a fully qualified name of a Java class that implements the interface org.eclipse.debug.ui.console.IConsoleContentProvider.
