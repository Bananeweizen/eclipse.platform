Debug Platform User-Interface

The platform debug UI plug-in, org.eclipse.debug.ui, implements a debug user-interface, allowing the user to view and manipulate debug targets, processes, threads, stack frames, and variables. It provides standard debug actions such as stepping, terminating, etc.

Extension Points

The following extension points are used to support and extend the debug UI.:

  • org.eclipse.debug.ui.debugActionGroups
  • org.eclipse.debug.ui.debugModelPresentations
  • org.eclipse.debug.ui.launchConfigurationTabGroups
  • org.eclipse.debug.ui.launchConfigurationTabs
  • org.eclipse.debug.ui.launchConfigurationTypeImages
