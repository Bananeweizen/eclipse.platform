Java editing support is provided by the Eclipse Java Tooling. The File Viewer example shows how a simple application can be implemented using SWT. This application provides the ability to navigate files and folders and manipulate them via Drag and Drop. It includes the use of alternate threads for long actions and demonstrates the use of the Tree, Table, Toolbar and Program.

Running the example

Follow the Manual Setup instructions to install and run the example from your workspace.

The "Main" class is org.eclipse.swt.examples.fileviewer.FileViewer.

This example can also be run using the Example Launcher. Select the File Viewer item from the Standalone category and click Run.
