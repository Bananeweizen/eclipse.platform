Identifier: org.eclipse.team.core.providers

Description: This extension point is used to register a provider nature with the Team Core plugin. Registered provider natures benefit from the services provided by the Team Core plug-in. These include programmatic configuration of providers and cardinality enforcement (e.g. only one team nature can be assigned to a project at a time).

Configuration Markup:

<!ELEMENT perspective (description?)>
   <!ATTLIST perspective
      natureId  CDATA #REQUIRED
    >

natureId - the nature identifier to register

Examples: The following is an example of a providers extension:

<extension id="CVSProvider" point="org.eclipse.team.core.providers">
     <providers natureId="org.eclipse.team.provider.cvs.core.cvsnature"/>
 </extension>
